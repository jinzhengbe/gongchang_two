# 开发文档

## 项目说明
本文档用于记录项目的开发过程和重要信息。

## 发布流程
项目使用 `publish.sh` 脚本进行一键发布，该脚本会自动完成以下操作：
1. 更新开发文档
2. 更新开发日志
3. 提交代码到 Git 仓库

### 使用方法
1. 确保脚本具有执行权限：
```bash
chmod +x publish.sh
```

2. 执行发布脚本：
```bash
./publish.sh
```

### 脚本功能说明
- 自动记录当前日期
- 更新开发文档和日志
- 自动提交代码到 Git 仓库
- 使用彩色输出提示当前执行的操作
